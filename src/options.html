Translate Backlog & Redmine

Lưu cấu hình Redmine, Backlog và AI để extension có thể dịch và đồng bộ comment qua lại giữa 2 hệ thống.

💎 Cấu hình Redmine - Click để thu gọn/mở rộng

💡 Redmine Domain được cấu hình sẵn.

🔑 Lấy API Key tại: Tài khoản cá nhân (Bấm vào "Hiện" tại mục "Mã truy cập API").

💡 Project chứa các issue Report (dùng cho chức năng "Log Time Tháng").

Cấu hình Custom Fields (JSON Mapping)

💡 Nhập mã ID của các trường bắt buộc. Ví dụ: { "Severity": 46, "Role": 11, "QC Activity": 8 }

📝 Cấu hình Backlog (Thu gọn) - Click để mở rộng

💡 Mặc định: https://shift7.backlog.com

🔑 Lấy tại: Tài khoản > API trong Backlog của bạn. Cần thiết để gửi nội dung ngược lại từ Redmine sang Backlog.
