⚠️ HƯỚNG DẪN THIẾT LẬP OAUTH2 (CẦN LÀM 1 LẦN)

Vì chính sách bảo mật Google, Chrome Extension bắt buộc phải khai báo Client ID trong file manifest.json.

  1. Truy cập Google Cloud Console, tạo/chọn project.
  2. Vào Library → Enable Google Sheets API.
  3. Vào Credentials → Click Create Credentials > OAuth client ID.
  4. Chọn loại Chrome Extension. Copy chính xác ID extension này:
    đang tải...
  5. Click Create → Copy Client ID nhận được.
  6. Mở file manifest.json trong code của extension và cập nhật block sau: 
    đang tải...
  7. Sau khi lưu file manifest.json, hãy vào chrome://extensions và click nút Reload extension!
